# Mohammed Nasrullah Portfolio — Project Specification

## Objective

Production-ready portfolio website built from scratch with original premium design.

Content sourced from `old_portfolio.html`. Design inspired by [nithishvaduganathan.tech](https://nithishvaduganathan.tech) layout, spacing, and interactions — not a clone.

---

## Theme

| Token | Value |
|-------|-------|
| Background | `#a4f9fd` |
| Cards | `#ffffff` |
| Primary Text | `#000000` |
| Accent | `#000000` |
| Borders | `rgba(0,0,0,0.15)` |
| Shadow | Soft modern shadow |

No gradients. No dark mode. Minimal and elegant.
